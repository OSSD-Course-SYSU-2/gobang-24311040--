# 五子棋游戏 (Gobang)

一个基于HarmonyOS开发的五子棋游戏应用，支持双人对战和人机对战模式。

## 功能特性

### 🎮 游戏模式
- **双人对战 (PVP)**: 两人轮流落子
- **人机对战 (PVE)**: 与AI进行对战

### 🎯 核心功能
- **15×15标准棋盘**: 符合标准五子棋规则
- **智能AI对手**: 基于评分算法的AI，具备攻防策略
- **胜负判定**: 自动检测五子连珠，支持四个方向判定
- **悔棋功能**: 支持撤销上一步操作
- **游戏重置**: 一键重新开始游戏
- **先后手切换**: 可交换黑白棋子

### 📊 统计功能
- 黑方胜局统计
- 白方胜局统计
- 和棋统计
- 总局数统计

### 🎨 界面特性
- 原木色棋盘背景
- 黑白棋子区分明显
- 最后落子标记
- 胜利连线高亮显示
- 星位点标记

## 技术实现

### Canvas绘制
- 使用HarmonyOS Canvas API绘制棋盘
- 动态计算棋盘尺寸和格子大小
- 支持响应式布局

### AI算法
- **位置评估**: 对每个空位进行评分
- **连子检测**: 检测活四、冲四、活三等棋型
- **攻防策略**: 同时考虑进攻和防守得分
- **中心优势**: 中心位置和星位点加分

### 胜负判定
- 四方向检测：水平、垂直、正斜、反斜
- 五子连珠判定
- 和棋判定（棋盘填满）

## 使用说明

1. **开始游戏**: 启动应用后自动开始新游戏
2. **落子**: 点击棋盘空位放置棋子
3. **切换模式**: 点击"切换到人机/双人"按钮
4. **悔棋**: 点击"悔棋"按钮撤销上一步
5. **重置**: 点击"重置"按钮重新开始
6. **交换先后手**: 点击"交换先后手"按钮
