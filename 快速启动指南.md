# 五子棋游戏 - 快速启动指南

## 🚀 如何运行游戏

### 方法一：在DevEco Studio中运行
1. 打开 DevEco Studio
2. 导入项目：`D:\gobang`
3. 等待项目加载完成
4. 点击工具栏的 **运行按钮** (绿色三角形)
5. 选择模拟器或真机设备
6. 应用将自动安装并启动

### 方法二：命令行编译安装
```bash
# 编译项目
hvigorw assembleHap

# 安装到设备
hdc install entry/build/default/outputs/default/entry-default-signed.hap
```

## 🎮 游戏操作说明

### 启动流程
1. 应用启动 → 显示"五子棋"加载页面
2. 自动跳转 → 进入游戏主界面
3. 显示棋盘 → 可以开始游戏

### 基本操作
- **落子**：点击棋盘空位
- **悔棋**：点击"悔棋"按钮
- **重置**：点击"重置"按钮
- **切换模式**：点击"人机模式/双人模式"按钮
- **交换先手**：点击"交换先手"按钮

### 游戏模式
- **双人对战**：两人轮流点击落子
- **人机对战**：玩家执黑，AI执白

## 🎯 游戏规则
- 黑棋先行
- 五子连珠获胜（横、竖、斜均可）
- 棋盘填满为和棋

## 📱 界面说明
- **顶部**：标题栏
- **状态栏**：显示当前回合和游戏模式
- **中间**：15×15棋盘（Canvas绘制）
- **底部**：控制按钮和统计信息

## ⚠️ 如果遇到问题

### 问题1：显示白屏
**解决方案**：
- 确保已重新编译项目
- 清理项目缓存：Build → Clean Project
- 重新运行应用

### 问题2：棋盘不显示
**解决方案**：
- 检查Canvas是否正确初始化
- 确保设备支持Canvas API
- 查看日志输出是否有错误

### 问题3：无法点击落子
**解决方案**：
- 确保点击在棋盘范围内
- 检查游戏是否已结束
- AI思考时无法落子（人机模式）
